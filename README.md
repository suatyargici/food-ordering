## Scripts

Run the development server

```bash
yarn dev
```

Building for production

```bash
yarn build
```

Run in production mode

```bash
yarn start
```

Run Storybook server

```bash
yarn storybook
```

Build Storybook as a static web application

```bash
yarn build-storybook
```
